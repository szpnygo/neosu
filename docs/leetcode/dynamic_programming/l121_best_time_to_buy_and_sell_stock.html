121.买卖股票的最佳时机

2021-05-09
2分钟阅读时长
算法 , LeetCode
动态规划

题目

给定一个数组 $prices$ ，它的第 $i$ 个元素 $prices[i]$ 表示一支给定股票第 $i$ 天的价格。

你只能选择 某一天 买入这只股票，并选择在 未来的某一个不同的日子 卖出该股票。设计一个算法来计算你所能获取的最大利润。

返回你可以从这笔交易中获取的最大利润。如果你不能获取任何利润，返回 0 。

示例1：

输入：[7,1,5,3,6,4]
输出：5
解释：
在第2天（股票价格 = 1）的时候买入，在第5天（股票价格 = 6）的时候卖出
最大利润 = 6-1 = 5
注意利润不能是 7-1 = 6, 因为卖出价格需要大于买入价格；
同时，你不能在买入前卖出股票。

示例2：

输入：prices = [7,6,4,3,1]
输出：0
解释：在这种情况下, 没有交易完成, 所以最大利润为 0。

提示：

  • $1 <= prices.length <= 10^5$
  • $0 <= prices[i] <= 10^4$

题解

func maxProfit(prices []int) int {
    min := math.MaxInt32
    max := 0
    for _,p := range prices{
        if p < min{
            min = p
        } else if p - min > max{
            max = p - min
        }
    }
    return max
}

思路

利用动态规划去解此题

假设$f(i)$代表以第$i$天卖出股票的最大利润，第$i$天卖出要获取最大利润，那就是要在之前在最低价格的那天买入。

所以可以写作：

$f(i) = prices[i] - min(prices[0…i-1])$

最终的结果就是$max(f(0…i))$

把推导公式写成代码：

func maxProfit(prices []int) int {
	min := prices[0] //第一天作为第一个最小价格
	max := 0
    //因为是买卖，第一天只能买，不能卖
	for i := 1; i < len(prices); i++ {
		if prices[i-1] < min {  //记录下之前价格最小的值
			min = prices[i-1]
		}
		result := prices[i] - min //计算f(i)的值
		if result > max { //取最大的f(i)作为最终结果
			max = result
		}
	}
	return max
}


//可以再简化一下，性能没什么变化，只是看着简洁一些
func maxProfit(prices []int) int {
    min := math.MaxInt32
    max := 0
    for _,p := range prices{
        if p < min{
            min = p
        } else if p - min > max{ //p小于min了，p-min肯定是负数，所以这里可以用else
            max = p - min
        }
    }
    return max
}
