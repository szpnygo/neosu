152.乘积最大子数组

2021-05-12
2分钟阅读时长
算法 , LeetCode
动态规划

题目

给你一个整数数组$nums$ ，请你找出数组中乘积最大的连续子数组（该子数组中至少包含一个数字），并返回该子数组所对应的乘积。

示例1：

输入: [2,3,-2,4]
输出: 6
解释: 子数组 [2,3] 有最大乘积 6。

示例2：

输入: [-2,0,-1]
输出: 0
解释: 结果不能为 2, 因为 [-2,-1] 不是子数组。

题解

func maxProduct(nums []int) int {
	maxF, minF, result := nums[0], nums[0], nums[0]
	for i := 1; i < len(nums); i++ {
		ma, mi := maxF, minF
		maxF = max(ma*nums[i], max(mi*nums[i], nums[i]))
		minF = min(mi*nums[i], min(ma*nums[i], nums[i]))
		result = max(maxF, result)
	}
	return result
}

func max(a, b int) int {
	if a > b {
		return a
	}
	return b
}

func min(a, b int) int {
	if a > b {
		return b
	}
	return a
}

思路

利用动态规划去解此题

此题类似于 53.最大子序和

继续按照动态规划的思路去思考

我们假设$f(i)$为以$nums[i]$结尾的乘积最大子数组

我们首先可能会想到，$f(i)$ 等于 $nums[i]$本身和$f(i-1) * nums[i]$最大的一个

$f(i) = max(f(i-1) * nums[i], nums[i])$

这个53.最大子序和基本一样

但是这样会有一个问题，因为此题是乘积，并且有负数

我们知道负负得正，而这个思路里，我们抛弃了负数，在负负等正的情况下，$f(i-1)$越小越好，也就是$min$

所以我们不仅要记录下最大的，还要记录最小的

$f_{max}(i) = max(f_{max}(i) * nums[i], nums[i])$

$f_{min}(i) = min(f_{min}(i) * nums[i], nums[i])$

$result = max(f_{max}(i))(i..n)$

根据公式编写代码

func maxProduct(nums []int) int {
    maxdp := make([]int, len(nums))
    mindp := make([]int, len(nums))
    maxdp[0] = nums[0]
    mindp[0] = nums[0]
    result := nums[0]
    for i := 1; i < len(nums);i++{
        // 如果 num[i] < 0, 也就是负数，我们此时如果要得到最大值
        // 就是期望要乘的值也是负数，并且越小越好
        // 所以这里是 mindp[i-1] * nums[i]
        if nums[i] < 0{
            maxdp[i] = max(mindp[i-1] * nums[i], nums[i])
            mindp[i] = min(maxdp[i-1] * nums[i], nums[i])
        }else{
            maxdp[i] = max(maxdp[i-1] * nums[i], nums[i])
            mindp[i] = min(mindp[i-1] * nums[i], nums[i])
        }
        //每次计算最大值
        result = max(maxdp[i], result) 
    }
    return result
}

        return a
    }
    return b
