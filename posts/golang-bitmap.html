Golang实现简单BitSet(一)

2021-05-10
3分钟阅读时长
Golang

Bitmap (位图)是一个十分有用的数据结构。所谓的 Bit-map 就是用一个 bit 位来标记某个元素对应的 Value，而 Key 即是该元素。由于采用了 Bit 为单位来存储数据，因此在内存占用方面，可以大大节省。

最近刚学了Redis的bitmap位图，如Java语言本身就自带了BitSet，而Golang中并没有相关实现。所以就自己手写了一个简单的Bitset位图。

自己实现一遍后，虽然还不是很完善，但是还是挺简单的。后续有时间会继续完成。加入扩容，统计之类的。

BitSet基本原理

使用位数组来记录数据，在每个位上用0和1表示相关数据，1存在，0不存在。好处是可以节省大量的空间。

如1亿用户，只需要$100000000 / 8 / 1024 / 1024 ≈ 12mb$

BitSet实现

Golang本身没有BitSet这种类型，我们可以自行构造实现。

使用$[]uint64$来储存位数组，其中$uint64$是64位的。所以在储存和使用的时候还需要做一些处理。

定义结构

首先我们定义一些常量方便使用，$shift$和$mask$，后续到使用的时候会介绍如何使用

定义一个$struct$，其他$data$的类型是$[]uint64$

创建一个函数来生成，$data$的长度是$size»shift$，也就是 $size / 64$

因为我们使用的是位，而一个$uint64$是64位，所以初始化的时候除64。

const (
	shift    = 6    // 2^6 = 64
	mask     = 0x3f // 63
)

type BitSet struct {
	data []uint64 //64位
}

//创建BitSet
func NewBitSet(size int) *BitSet {
	return &BitSet{
		data: make([]uint64, size>>shift+1),
	}
}

Get方法

里面涉及到了2个位运算，都不复杂

index := n >> shift

对n进行位运算，相当于 $n / 2^6$，因为我们输入的偏移量以位作为偏移的，我们储存的是uint64，64位为一个单位。

所以要先计算出，他位于哪一个uint64。

然后再计算在位中的偏移位置，就是对64取余。

n&mask就等于n%64。可以解释一下，比如

例如 789 & 63 = 789 % 64 = 21
789 二进制    0011 0001 0101
63  二进制    0000 0011 1111
&的后的结果    0000 0001 0101

然后再对1左移n&mask位，获得偏移位置，通过&求值，如果等于0，该位置值是0，否则是1

set.data[index]&(1<<uint(n&mask))

完整方法

func (set *BitSet) Get(n int) int {
	if n < 0 { // n必须是正数
		return -1
	}
	index := n >> shift
	if index >= len(set.data) {
		return -1 //防止溢出
	}
	if set.data[index]&(1<<uint(n&mask)) == 0 {
		return 0
	}
	return 1
}

Set方法

同样是通过位运算，设置值。其中的位运算大家可以自己研究一下。

func (set *BitSet) Set(n int, value int) int {
	if n < 0 || !(value == 0 || value == 1) {
		return -1 //value只能是0或1
	}
	index := n >> shift
	if index >= len(set.data) {
		return -1 //溢出
	}
	oldValue := set.Get(n) //模仿redis bitmap，设置后返回设置前的值
	if oldValue == 0 && value == 1 {
		set.data[index] |= 1 << uint(n&mask) //对应的位设置为1，直接安位或操作即可
	} else if oldValue == 1 && value == 0 {
		set.data[index] &^= 1 << uint(n&mask) //对应的位设置为0，先按位取反，然后进行与操作
	}
	return oldValue
}

func main() {
	bitSet := NewBitSet(10000)
	bitSet.Set(1234, 1)
	fmt.Println(bitSet.Get(1234)) //1
	fmt.Println(bitSet.Get(1235)) //0
	bitSet.Set(1234, 0)
	fmt.Println(bitSet.Get(1234)) //0
}
